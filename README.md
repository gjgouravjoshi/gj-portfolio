# GJ Portfolio

Static portfolio site for Gourav Joshi.

## Deploy on Render

1. Push this folder to a GitHub repository.
2. In Render, choose **New > Static Site**.
3. Connect the GitHub repository.
4. Use these settings if Render asks:
   - Build Command: `echo "Static site ready"`
   - Publish Directory: `.`

The `render.yaml` file includes the same static-site setup for Blueprint deploys.
